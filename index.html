Output: featured project

One lead pipeline. Built three times, on purpose.

To prove the logic — not the tool — was the real skill, I built the same CRM lead automation end-to-end in Zapier, Make.com, and n8n. Same trigger, same outcome, three different engines.

Form submitted→ Create/update contact→ Create company→ Create deal→ Send confirmation email→ Log to CRM database

Linear and dependable. A five-step Zap: new Google Forms response creates a HubSpot contact, creates the company, sends a Gmail confirmation, and logs the row to Google Sheets. Built for a team that wanted something simple to hand off and maintain.

Same pipeline, more resilience. The Make.com version adds duplicate-contact checking, retry-on-fail logic, and a separate logging branch for duplicate leads — so the workflow degrades gracefully instead of failing silently.

Same pipeline, more intelligence. In n8n, the workflow scores each lead on budget and job seniority, branches qualified vs. unqualified leads, and routes hot leads straight to a Slack alert for sales — built and tested with 197k+ community-verified logic.
